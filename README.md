# 《人类群星闪耀时》系列

> 十三部书稿，一个追问：人类如何理解、建立、信任、突破和重建系统？

---

## 这个仓库是什么

这是一个跨学科写作项目。十三本独立书稿从数学、计算、密码学、网络安全、宇宙学、地球科学、艺术史、商业史、科学方法论等方向，逼近同一个核心问题——**系统的构建与脆弱性**。

所有书稿均为个人独立写作，非翻译、非汇编。

---

## 如果只读三本

1. **《不确定性的尊严》** — 学会在不确定性中做判断
2. **《无形之战》** — 理解每一个系统都有未被预见的裂缝
3. **《可视的尊严》** — 感受人类如何让不可见之物变得可见
